### Почему у критерия Джини 2 формулы?

1. **Gini Impurity (Неопределенность Джини)** — используется в деревьях решений (CART).
   Формула: $$G = 1 - \sum_{i=1}^{n} p_i^2$$
   Где $p_i$ — вероятность (доля) класса $i$ в узле. Она показывает, какова вероятность ошибки, если мы случайным образом выберем элемент и назначим ему случайную метку в соответствии с распределением в узле.

2. **Gini Index / Coefficient (Коэффициент Джини)** — используется в экономике для измерения неравенства доходов.
   Обычно выражается через площадь между кривой Лоренца и линией равенства. 
   Его формула часто выглядит как отношение: $$G = \frac{A}{A + B}$$

Формулу Неопределенности Джини можно раскрыть через сумму произведений вероятностей разных классов:
$$G = \sum_{i \neq j} p_i p_j$$
